# JAAQ SDK Examples

This folder contains comprehensive examples demonstrating the JAAQ SDK capabilities across different environments and use cases.

## Quick Links

- [Basic Node.js Examples](./basic/) - Get started with Node.js (ESM and CJS)
- [Browser Examples](./browser/) - CDN and vanilla JavaScript examples
- [React + Vite Example](./react-vite/) - Modern React application with TypeScript

## Prerequisites

All examples require:

- JAAQ API Key
- JAAQ Client ID

Get your credentials from the JAAQ dashboard.

## Examples Overview

### Basic (Node.js)

Simple Node.js scripts demonstrating core SDK functionality:

- ESM and CommonJS support
- Video retrieval
- Collections listing
- Error handling

### Browser

Vanilla JavaScript examples using the SDK via CDN or local bundle:

- Basic API calls with HTML
- Video player integration
- HLS streaming support
- Advanced UI components (gallery, playlist)

### React + Vite

Full-featured React application showcasing:

- TypeScript integration
- Component-based architecture
- Video player components
- Collections management
- HLS streaming with hls.js

## Getting Started

1. Choose an example folder based on your use case
2. Copy `.env.example` to `.env` and add your credentials
3. Follow the README in each folder for specific setup instructions
