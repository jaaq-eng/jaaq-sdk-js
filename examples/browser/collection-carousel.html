JAAQ SDK - Collection Carousel

Video collection carousel with Splide.js integration

Features: This example demonstrates a responsive video carousel that fetches collections and displays videos with a center-focused layout. The center video is larger, and the carousel adapts to different screen sizes (3 videos on desktop, 2 on tablet, 1 on mobile).

CDN Usage

Use this snippet to load the collection player from CDN:

<script src="https://cdn.jaaq.app/latest/ui/webcomponents/jaaq-webcomponents-bundled.min.js"></script>

<jaaq-collection-player 
  collection-id="your-collection-id" 
  api-key="your-api-key" 
  client-id="your-client-id">
</jaaq-collection-player>
