JAAQ SDK - Web Component Advanced

Programmatic control and state management

Advanced Features: This example demonstrates programmatic control of the web component, including property setters, method calls, and state inspection. Components are auto-registered on import.

CDN Usage

Use this snippet to load the player from CDN without building:

<script src="https://cdn.jaaq.app/latest/ui/webcomponents/jaaq-webcomponents-bundled.min.js"></script>

<jaaq-video-player 
  video-id="your-video-id" 
  api-key="your-api-key" 
  client-id="your-client-id">
</jaaq-video-player>
