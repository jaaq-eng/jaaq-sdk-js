JAAQ SDK - Multiple Vanilla UI Players

Demonstration of multiple embeddable video players on the same page

Multiple Players Support: This example demonstrates declarative player initialization with multiple players. CSS is only injected once and all player instances work correctly on the same page without conflicts. Players 1 & 2 use the declarative approach, while Player 3 uses manual instantiation.

CDN Usage

Use this snippet to load the vanilla UI player from CDN without building:

<script src="https://cdn.jaaq.app/jaaq-sdk-js/latest/ui/jaaq-ui-bundled.min.js"></script>

<div id="player"></div>

<script>
  const player = new JaaqUI.JaaqVideoPlayer(
    document.getElementById('player'),
    {
      apiKey: 'your-api-key',
      clientId: 'your-client-id',
      videoId: 'your-video-id',
      autoplay: false
    }
  );
  
  player.on('loaded', (video) => console.log(video));
</script>
