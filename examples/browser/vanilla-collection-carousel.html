JAAQ Vanilla JS - Collection Carousel

Pure JavaScript implementation with Splide.js integration

Features:

- Event emitter pattern with on() and off() methods

- Programmatic API: next(), prev(), go(index), refresh()

- Responsive: 3 videos on desktop, 2 on tablet, 1 on mobile

Minimal Example

<script src="jaaq-vanilla-bundled.min.js"></script>
<div id="carousel"></div>
<script>
  const { JaaqCollectionPlayer, registerJaaqComponents } = JaaqVanilla;
  
  registerJaaqComponents();

  const carousel = new JaaqCollectionPlayer('#carousel', {
    collectionId: 'your-collection-id',
    apiKey: 'your-api-key',
    clientId: 'your-client-id',
    autoplay: true
  });
</script>
