JAAQ SDK - Declarative Web Component

Zero JavaScript Required! Just add the custom element to your HTML with attributes. The component will automatically initialize when the page loads.

How to Use

1. Include the script (auto-registers components on load):

<script src="https://cdn.jaaq.com/webcomponents.js"></script>

2. Add the custom element to your HTML:

<jaaq-video-player video-id="your_video_id" api-key="your_api_key" client-id="your_client_id" autoplay="false" width="100%"> </jaaq-video-player>

That's it! No JavaScript code needed.

Note: Replace the attribute values below with your actual credentials to see the player in action.

Available Attributes

  • video-id - (required) The ID of the video to play
  • api-key - (optional) Your JAAQ API key
  • client-id - (optional) Your client ID
  • base-url - (optional) Custom API base URL
  • autoplay - (optional) Auto-play the video (true/false)
  • width - (optional) Player width (e.g., "100%", "800px")
  • height - (optional) Player height (e.g., "auto", "450px")

Custom Events

Listen to player events with standard event listeners:

const player = document.querySelector('jaaq-video-player'); player.addEventListener('jaaq:loaded', (e) => { console.log('Video loaded:', e.detail); }); player.addEventListener('jaaq:play', () => { console.log('Video started playing'); }); player.addEventListener('jaaq:pause', () => { console.log('Video paused'); }); player.addEventListener('jaaq:error', (e) => { console.error('Error:', e.detail); });
