JAAQ SDK - Declarative Vanilla UI Player

Auto-initialized video player using data attributes

Declarative Approach: Simply add data attributes to a div element and the player will auto-initialize on page load. No manual JavaScript instantiation required!

How It Works

Add this HTML to your page:

<div data-jaaq-player
     data-api-key="your_api_key"
     data-client-id="your_client_id"
     data-video-id="your_video_id"
     data-autoplay="false"></div>

<script src="https://cdn.jaaq.com/player.js"></script>

Access the player instance programmatically:

const player = JaaqUI.JaaqPlayer.getPlayer('[data-video-id="your_video_id"]');
player.on('play', () => console.log('Playing'));
