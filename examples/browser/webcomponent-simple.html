JAAQ SDK - Web Component

Native custom element with Shadow DOM encapsulation

Note: This example uses native web components with the <jaaq-video-player> custom element. The component uses Shadow DOM for style encapsulation and provides a clean, framework-agnostic API.

How to Use

1. Include the script (auto-registers the component):

<script src="jaaq-webcomponents-bundled.min.js"></script>

2. Add the custom element to your HTML:

<jaaq-video-player
  video-id="your_video_id"
  api-key="your_api_key"
  client-id="your_client_id"
  autoplay="false">
</jaaq-video-player>
