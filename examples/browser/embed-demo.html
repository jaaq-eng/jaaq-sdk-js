JAAQ Embed Player - Demo

Learn how to embed the JAAQ video player in an iframe

Iframe Embed: Use the ui-vanilla-embed.html file as your iframe src with URL query parameters. The embed uses the <jaaq-video-player> web component internally and communicates with the parent page via postMessage for events and commands.

Local Testing: Run pnpm build:local from project root, then pnpm serve from examples/browser. The embed will be available at http://localhost:9999/jaaq-sdk-js/latest/embed.html

CDN Usage

Use this snippet to embed the player from CDN in production:

<iframe 
  src="https://cdn.jaaq.app/jaaq-sdk-js/latest/embed/embed.html?apiKey=YOUR_API_KEY&clientId=YOUR_CLIENT_ID&videoId=YOUR_VIDEO_ID"
  width="800" 
  height="450" 
  frameborder="0" 
  allowfullscreen>
</iframe>

Dynamic JavaScript Creation

const iframe = document.createElement('iframe');
iframe.src = `ui-vanilla-embed.html?apiKey=${apiKey}&clientId=${clientId}&videoId=${videoId}`;
iframe.width = '800';
iframe.height = '450';
iframe.frameBorder = '0';
iframe.allowFullscreen = true;
document.getElementById('container').appendChild(iframe);

Listening to Events (postMessage)

window.addEventListener('message', (event) => {
  if (event.data && event.data.type === 'jaaq-player-event') {
    console.log('Event:', event.data.event, event.data.data);
    
    switch (event.data.event) {
      case 'loaded':
        console.log('Video loaded:', event.data.data.title);
        break;
      case 'play':
        console.log('Video started playing');
        break;
      case 'pause':
        console.log('Video paused');
        break;
      case 'ended':
        console.log('Video ended');
        break;
    }
  }
});

Sending Commands to Player

const iframe = document.querySelector('iframe');

iframe.contentWindow.postMessage({
  type: 'jaaq-player-command',
  command: 'play'
}, '*');

iframe.contentWindow.postMessage({
  type: 'jaaq-player-command',
  command: 'setVolume',
  args: { volume: 0.5 }
}, '*');

iframe.contentWindow.postMessage({
  type: 'jaaq-player-command',
  command: 'seek',
  args: { time: 30 }
}, '*');

Available URL Parameters

apiKey      (required) - Your JAAQ API key
clientId    (required) - Your JAAQ client ID
videoId     (required) - Video ID to load
autoplay    (optional) - true/false, default: false
width       (optional) - Player width, default: 100%
height      (optional) - Player height, default: 100%
baseUrl     (optional) - Custom API base URL

Available Events

ready           - Player is ready
loaded          - Video metadata loaded
play            - Video started playing
pause           - Video paused
timeupdate      - Current time updated
volumechange    - Volume changed
ended           - Video finished playing
error           - Error occurred
fullscreenchange - Fullscreen toggled

Available Commands

play        - Start playback
pause       - Pause playback
toggleMute  - Toggle mute
setVolume   - Set volume (args: { volume: 0-1 })
seek        - Seek to time (args: { time: seconds })
getState    - Get current player state
